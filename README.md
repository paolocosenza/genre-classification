# music-genre-classification

Web app: https://paolocosenza-music-genre-classifica-genre-classification-vdc16o.streamlit.app/

Web application project containing a simple music genre classifier trained on the <a href=https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification> GTZAN Dataset</a>. It takes the URL of a YouTube song and classifies its genre.
